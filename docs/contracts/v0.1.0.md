## Philosophy & Rules

1. **API-first with context.** Every crate publishes its final public interface before implementation begins.
2. **`TyCtx` is the spine.** Post-typeck, `TyCtx` is frozen, `Send + Sync`, passed as `&TyCtx`. No `&mut TyCtx` beyond typeck.
3. **Real interning.** All Chalk associated types are `u32` indices into arenas. `Substitution` is interned. No `Box`/`Vec` pretending to intern.
4. **Tracing mandated.** Every public function in a logic crate carries `#[tracing::instrument(skip(...))]`.
5. **Stubs allowed, mock-driven.** Implementations may begin as `todo!()`. Each crate supports a `mock` feature.
6. **Contract governance.** Changes to public types require a written "Contract Change Request" reviewed by at least one other track lead.
7. **Minimal v0.1.0.** Generators, virtual dispatch shims, coverage, retag, and niche optimizations are excluded.
8. **No `unsafe` without a safety proof.** Every `unsafe` block has a `// SAFETY:` comment.
9. **Primitives isolation.** `glyim-primitives` is the only shared enum crate. Frontend crates never depend on `glyim-type`.

// crates/glyim-arena/src/lib.rs
//! Typed index types and arena allocators.
//!
//! `IndexVec` provides bounds-checked indexing in debug builds
//! and `try_index` for fallible access. The `define_idx!` macro
//! generates strongly-typed index newtypes.

use std::fmt;
use std::marker::PhantomData;
use std::ops::{Index, IndexMut};

// ── Idx ──

/// A strongly-typed index into an `IndexVec<I, T>`.
/// Zero-cost newtype over `u32`.
#[derive(Clone, Copy, PartialEq, Eq, Hash, PartialOrd, Ord)]
pub struct Idx<T> {
    pub raw: u32,
    _marker: PhantomData<T>,
}

impl<T> fmt::Debug for Idx<T> {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "Idx({})", self.raw)
    }
}

impl<T> Idx<T> {
    #[inline]
    pub fn from_raw(raw: u32) -> Self {
        Self { raw, _marker: PhantomData }
    }

    #[inline]
    pub fn index(self) -> usize {
        self.raw as usize
    }
}

/// Trait for types that can be used as `IndexVec` indices.
///
/// The `define_idx!` macro emits `$crate::IdxLike` so it resolves
/// correctly regardless of which crate invokes it.
pub trait IdxLike: Copy + Eq + fmt::Debug + 'static {
    fn from_raw(raw: u32) -> Self;
    fn to_raw(self) -> u32;
    fn index(self) -> usize {
        self.to_raw() as usize
    }
}

impl<T> IdxLike for Idx<T> {
    fn from_raw(raw: u32) -> Self {
        Idx::from_raw(raw)
    }
    fn to_raw(self) -> u32 {
        self.raw
    }
}

// ── define_idx! macro ──

/// Define a strongly-typed index wrapper.
///
/// Generates a newtype struct, `from_raw`, `index`, and an `IdxLike` impl.
/// The macro emits `$crate::IdxLike` so it resolves correctly
/// regardless of which crate invokes it.
#[macro_export]
macro_rules! define_idx {
    ($name:ident) => {
        #[derive(Clone, Copy, Debug, PartialEq, Eq, Hash, PartialOrd, Ord)]
        pub struct $name(pub u32);

        impl $name {
            #[inline]
            pub fn from_raw(raw: u32) -> Self {
                Self(raw)
            }

            #[inline]
            pub fn index(self) -> usize {
                self.0 as usize
            }
        }

        impl $crate::IdxLike for $name {
            fn from_raw(raw: u32) -> Self {
                Self(raw)
            }
            fn to_raw(self) -> u32 {
                self.0
            }
        }
    };
}

// ── IndexVec ──

/// A vector indexed by `I` instead of `usize`.
///
/// Indexing panics with a descriptive message in debug builds.
/// Use `try_index` for fallible access.
#[derive(Clone, Debug)]
pub struct IndexVec<I: IdxLike, T> {
    raw: Vec<T>,
    _marker: PhantomData<I>,
}

impl<I: IdxLike, T> IndexVec<I, T> {
    // ── Constructors ──

    pub fn new() -> Self {
        Self { raw: Vec::new(), _marker: PhantomData }
    }

    pub fn with_capacity(cap: usize) -> Self {
        Self { raw: Vec::with_capacity(cap), _marker: PhantomData }
    }

    pub fn from_raw(raw: Vec<T>) -> Self {
        Self { raw, _marker: PhantomData }
    }

    // ── Mutation ──

    pub fn push(&mut self, val: T) -> I {
        let idx = I::from_raw(self.raw.len() as u32);
        self.raw.push(val);
        idx
    }

    pub fn reserve(&mut self, additional: usize) {
        self.raw.reserve(additional);
    }

    // ── Access ──

    pub fn len(&self) -> usize {
        self.raw.len()
    }

    pub fn is_empty(&self) -> bool {
        self.raw.is_empty()
    }

    /// Fallible indexing — returns `None` on out-of-bounds.
    pub fn try_index(&self, idx: I) -> Option<&T> {
        let i = idx.index();
        if i < self.raw.len() {
            Some(&self.raw[i])
        } else {
            None
        }
    }

    /// Fallible mutable indexing.
    pub fn try_index_mut(&mut self, idx: I) -> Option<&mut T> {
        let i = idx.index();
        if i < self.raw.len() {
            Some(&mut self.raw[i])
        } else {
            None
        }
    }

    pub fn get(&self, idx: I) -> Option<&T> {
        self.try_index(idx)
    }

    pub fn get_mut(&mut self, idx: I) -> Option<&mut T> {
        self.try_index_mut(idx)
    }

    // ── Iteration ──

    pub fn iter(&self) -> impl Iterator<Item = &T> {
        self.raw.iter()
    }

    pub fn iter_mut(&mut self) -> impl Iterator<Item = &mut T> {
        self.raw.iter_mut()
    }

    pub fn iter_enumerated(&self) -> impl Iterator<Item = (I, &T)> {
        self.raw
            .iter()
            .enumerate()
            .map(|(i, v)| (I::from_raw(i as u32), v))
    }

    pub fn into_iter_enumerated(self) -> impl Iterator<Item = (I, T)> {
        self.raw
            .into_iter()
            .enumerate()
            .map(|(i, v)| (I::from_raw(i as u32), v))
    }

    // ── Conversion ──

    pub fn into_raw(self) -> Vec<T> {
        self.raw
    }

    pub fn as_slice(&self) -> &[T] {
        &self.raw
    }

    pub fn as_mut_slice(&mut self) -> &mut [T] {
        &mut self.raw
    }

    pub fn last(&self) -> Option<&T> {
        self.raw.last()
    }
}

impl<I: IdxLike, T> Default for IndexVec<I, T> {
    fn default() -> Self {
        Self::new()
    }
}

/// Indexing panics with a descriptive error message.
impl<I: IdxLike, T> Index<I> for IndexVec<I, T> {
    type Output = T;

    fn index(&self, idx: I) -> &T {
        let i = idx.index();
        assert!(
            i < self.raw.len(),
            "IndexVec index out of bounds: {:?} (raw={}) >= length {} (index type: {})",
            idx,
            i,
            self.raw.len(),
            std::any::type_name::<I>()
        );
        &self.raw[i]
    }
}

impl<I: IdxLike, T> IndexMut<I> for IndexVec<I, T> {
    fn index_mut(&mut self, idx: I) -> &mut T {
        let i = idx.index();
        assert!(
            i < self.raw.len(),
            "IndexVec index out of bounds: {:?} (raw={}) >= length {} (index type: {})",
            idx,
            i,
            self.raw.len(),
            std::any::type_name::<I>()
        );
        &mut self.raw[i]
    }
}

// ── EmptyIter helper ──

/// Produces an empty iterator with the same element type as a given IndexVec.
/// Useful for default implementations that return no results.
pub fn empty_iter<T>() -> impl Iterator<Item = T> {
    std::iter::empty()
}

// crates/glyim-interner/src/lib.rs
//! Thread-safe string interning.
//!
//! `Interner` provides write capability (intern new strings).
//! `Resolver` is cheaply clonable, `Send + Sync`, read-only.
//! Well-known keyword symbols are pre-interned via `Kw`.

pub use lasso::Spur as Symbol;

/// An interned identifier. Copy + Hash + Eq in O(1).
pub type Name = Symbol;

// ── Write capability ──

/// Thread-safe string interner. Owns the allocation.
pub struct Interner {
    inner: lasso::ThreadedRodeo,
}

/// Read-only resolver. Cheaply clonable, `Send + Sync`.
#[derive(Clone)]
pub struct Resolver {
    inner: lasso::ThreadedRodeo,
}

impl Interner {
    pub fn new() -> Self {
        Self {
            inner: lasso::ThreadedRodeo::new(),

/// A contiguous source span. 16 bytes, `Copy`.
/// The `ctx` field tracks hygiene/macro expansion context.
#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash)]
pub struct Span {
    pub file: FileId,
    pub lo: ByteIdx,
    pub hi: ByteIdx,
    pub ctx: SyntaxContext,
}

impl Span {
    pub const DUMMY: Span = Span {
        file: FileId::BOGUS,
        lo: ByteIdx(0),
        hi: ByteIdx(0),
        ctx: SyntaxContext::ROOT,
    };

    pub fn new(file: FileId, lo: ByteIdx, hi: ByteIdx, ctx: SyntaxContext) -> Self {
        debug_assert!(lo <= hi, "Span lo > hi: {:?} > {:?}", lo, hi);
        Self { file, lo, hi, ctx }
    }

    pub fn is_dummy(self) -> bool {
        self == Self::DUMMY
    }

    pub fn range(self) -> Range<usize> {
        self.lo.to_usize()..self.hi.to_usize()
    }

    /// Strip the hygiene context, yielding a plain source span.
    pub fn sans_ctx(self) -> Span {
        Span { ctx: SyntaxContext::ROOT, ..self }
    }

    /// Merge two spans into one covering both.
    /// Panics if they are from different files.
    pub fn to(self, other: Span) -> Span {
        debug_assert_eq!(self.file, other.file, "Cannot merge spans from different files");
        Span {
            file: self.file,
            lo: std::cmp::min(self.lo, other.lo),
            hi: std::cmp::max(self.hi, other.hi),
            ctx: SyntaxContext::ROOT,
        }
    }

    pub fn contains(self, other: Span) -> bool {
        self.file == other.file && self.lo <= other.lo && other.hi <= self.hi
    }

    pub fn len(self) -> u32 {
        self.hi.0.saturating_sub(self.lo.0)
    }
}

impl From<Span> for SourceSpan {
    fn from(s: Span) -> SourceSpan {
        SourceSpan::new(s.lo.0.into(), (s.hi.0.saturating_sub(s.lo.0)).into())
    }

/// and a final "too many errors" message is emitted.
pub struct DiagSink {
    diagnostics: Vec<GlyimDiagnostic>,
    error_count: usize,
    warning_count: usize,
    error_limit: usize,
    limited: bool,
}

impl DiagSink {
    pub fn new() -> Self {
        Self {
            diagnostics: Vec::new(),
            error_count: 0,
            warning_count: 0,
            error_limit: 50,
            limited: false,
        }
    }

    pub fn with_error_limit(limit: usize) -> Self {
        Self {
            diagnostics: Vec::new(),
            error_count: 0,
            warning_count: 0,
            error_limit: limit,
            limited: false,
        }
    }

    /// Emit a diagnostic. If the error limit is exceeded, subsequent
    /// errors are suppressed and a single "too many errors" message is added.
    pub fn emit(&mut self, diag: GlyimDiagnostic) {
        if diag.severity == DiagSeverity::Error {
            self.error_count += 1;
        }
        if diag.severity == DiagSeverity::Warning {
            self.warning_count += 1;
        }

        if self.error_count > self.error_limit {
            if !self.limited {
                self.diagnostics.push(GlyimDiagnostic::internal_error(format!(
                    "Too many errors emitted ({}); stopping now.",
                    self.error_count
                )));
                self.limited = true;
            }
            return;
        }

        self.diagnostics.push(diag);
    }

    pub fn has_errors(&self) -> bool {
        self.error_count > 0
    }

    pub fn error_count(&self) -> usize {
        self.error_count
    }

    pub fn warning_count(&self) -> usize {
        self.warning_count
    }

    pub fn was_limited(&self) -> bool {
        self.limited
    }

    pub fn into_diagnostics(self) -> Vec<GlyimDiagnostic> {
        self.diagnostics
    }

    pub fn diagnostics(&self) -> &[GlyimDiagnostic] {
        &self.diagnostics
    }
}

impl Default for DiagSink {
    fn default() -> Self {
        Self::new()
    }
}

impl Extend<GlyimDiagnostic> for DiagSink {
    fn extend<T: IntoIterator<Item = GlyimDiagnostic>>(&mut self, iter: T) {
        for diag in iter {
            self.emit(diag);
        }

// crates/glyim-vfs/src/lib.rs
//! Virtual file system.
//!
//! Abstracts over file I/O so the compiler never touches the disk
//! directly during compilation. File contents are stored in memory
//! and addressed by `FileId`. The VFS is `Send + Sync`.

use glyim_span::FileId;
use parking_lot::RwLock;
use std::path::{Path, PathBuf};

// ── Vfs ──

/// Virtual file system. Thread-safe via `RwLock`.
///
/// Files are addressed by `FileId`. The VFS owns the source text
/// and never reads from disk after initial load.
pub struct Vfs {
    files: RwLock<Vec<VfsFile>>,
    path_to_id: RwLock<std::collections::HashMap<PathBuf, FileId>>,
    next_id: RwLock<u32>,
}

#[derive(Clone, Debug)]
struct VfsFile {
    path: PathBuf,
    content: String,
}

impl Vfs {
    pub fn new() -> Self {
        Self {
            files: RwLock::new(Vec::new()),
            path_to_id: RwLock::new(std::collections::HashMap::new()),
            next_id: RwLock::new(0),
        }
    }

    /// Add a file from disk. Returns its `FileId`.
    /// If the file is already loaded, returns the existing `FileId`.
    #[tracing::instrument(skip(self))]
    pub fn add_file(&self, path: &Path) -> FileId {
        // Check if already loaded
        if let Some(&id) = self.path_to_id.read().get(path) {
            return id;
        }

        let content = std::fs::read_to_string(path)
            .unwrap_or_else(|e| {
                tracing::error!("Failed to read file {:?}: {}", path, e);
                String::new()
            });

        self.add_file_content(path, content)
    }

    /// Add a file with explicit content. Returns its `FileId`.
    /// Used in tests and LSP where the content comes from the editor.
    #[tracing::instrument(skip(self, content))]
    pub fn add_file_content(&self, path: &Path, content: String) -> FileId {
        // Check if already loaded
        if let Some(&id) = self.path_to_id.read().get(path) {
            return id;
        }

        let mut next_id = self.next_id.write();
        let id = FileId(*next_id);
        *next_id += 1;

        self.files.write().push(VfsFile {
            path: path.to_path_buf(),
            content,
        });
        self.path_to_id.write().insert(path.to_path_buf(), id);

        id
    }

    /// Update a file's content. Used by LSP on textDocument/didChange.
    #[tracing::instrument(skip(self, content))]
    pub fn set_file_content(&self, file_id: FileId, content: String) {
        let mut files = self.files.write();
        if let Some(file) = files.get_mut(file_id.0 as usize) {
            file.content = content;
        }
    }

    /// Get the content of a file.
    pub fn file_content(&self, file_id: FileId) -> Option<String> {
        self.files
            .read()
            .get(file_id.0 as usize)
            .map(|f| f.content.clone())
    }

    /// Get a reference to the content of a file. Avoids cloning.
    pub fn file_content_ref<R>(&self, file_id: FileId, f: impl FnOnce(&str) -> R) -> Option<R> {
        let files = self.files.read();
        files.get(file_id.0 as usize).map(|file| f(&file.content))
    }

    /// Get the path for a file.
    pub fn file_path(&self, file_id: FileId) -> Option<PathBuf> {
        self.files
            .read()
            .get(file_id.0 as usize)
            .map(|f| f.path.clone())
    }

    /// Resolve a `FileId` from a path.
    pub fn file_id(&self, path: &Path) -> Option<FileId> {
        self.path_to_id.read().get(path).copied()
    }

    /// Number of loaded files.
    pub fn len(&self) -> usize {
        self.files.read().len()
    }

    pub fn is_empty(&self) -> bool {
        self.files.read().is_empty()
    }
}

impl Default for Vfs {
    fn default() -> Self {
        Self::new()
    }
}

        (self as u16) >= Self::KwFn as u16 && (self as u16) <= Self::KwStatic as u16

    pub fn is_literal(self) -> bool {
        matches!(
            self,
            Self::IntLit | Self::FloatLit | Self::StringLit | Self::CharLit | Self::BoolLit

    pub fn is_node(self) -> bool {
        (self as u16) >= Self::SourceFile as u16 && (self as u16) < Self::Error as u16

        !self.is_node()

            // SAFETY: repr(u16) makes transmute sound IF raw is a valid discriminant.
            Some(unsafe { std::mem::transmute::<u16, Self>(raw) })
        } else {
            None

// crates/glyim-lex/src/lib.rs
//! Lexer: source text → token stream.
//!
//! Produces raw tokens with spans for the parser. Handles
//! keywords, literals, operators, punctuation, and trivia.

use glyim_syntax::SyntaxKind;
use glyim_span::{ByteIdx, FileId, Span};
use glyim_diag::GlyimDiagnostic;

// ── Token ──

/// A single lexical token with its span.
#[derive(Clone, Debug)]
pub struct Token {
    pub kind: SyntaxKind,
    pub span: Span,
    pub text: smol_str::SmolStr,
}

impl Token {
    pub fn new(kind: SyntaxKind, span: Span, text: impl AsRef<str>) -> Self {
        Self {
            kind,
            span,
            text: smol_str::SmolStr::from(text.as_ref()),
        }
    }
}

// ── LexResult ──

#[derive(Clone, Debug)]
pub struct LexResult {
    pub tokens: Vec<Token>,
    pub diagnostics: Vec<GlyimDiagnostic>,
}

// ── Lexer ──

pub struct Lexer<'a> {
    source: &'a str,
    file_id: FileId,
    pos: usize,
    diagnostics: Vec<GlyimDiagnostic>,
}

impl<'a> Lexer<'a> {
    pub fn new(source: &'a str, file_id: FileId) -> Self {
        Self {
            source,
            file_id,
            pos: 0,
            diagnostics: Vec::new(),
        }
    }

    fn span(&self, start: usize, end: usize) -> Span {
        Span::new(
            self.file_id,
            ByteIdx(start as u32),
            ByteIdx(end as u32),
            glyim_span::SyntaxContext::ROOT,
        )
    }

    fn peek(&self) -> Option<char> {
        self.source[self.pos..].chars().next()
    }

    fn peek_next(&self) -> Option<char> {
        let mut chars = self.source[self.pos..].chars();
        chars.next();
        chars.next()
    }

    fn advance(&mut self) -> Option<char> {
        let ch = self.source[self.pos..].chars().next();
        if let Some(c) = ch {
            self.pos += c.len_utf8();
        }
        ch
    }

    /// Lex the full source text into a token stream.
    #[tracing::instrument(skip(self))]
    pub fn lex(mut self) -> LexResult {
        let mut tokens = Vec::new();

        while self.pos < self.source.len() {
            let start = self.pos;

            // Skip trivia (whitespace and comments)
            if self.lex_trivia() {
                continue;
            }

            if self.pos >= self.source.len() {
                break;
            }

            let Some(ch) = self.peek() else { break };

            let kind = match ch {
                // Identifiers and keywords
                'a'..='z' | 'A'..='Z' | '_' => {
                    self.lex_ident_or_keyword();
                    let text = &self.source[start..self.pos];
                    let kind = lookup_keyword(text);
                    tokens.push(Token::new(kind, self.span(start, self.pos), text));
                    continue;
                }

                // Numeric literals
                '0'..='9' => {
                    self.lex_number();
                    let text = &self.source[start..self.pos];
                    tokens.push(Token::new(SyntaxKind::IntLit, self.span(start, self.pos), text));
                    continue;
                }

                // String literals
                '"' => {
                    self.lex_string();
                    let text = &self.source[start..self.pos];
                    tokens.push(Token::new(SyntaxKind::StringLit, self.span(start, self.pos), text));
                    continue;
                }

                // Char literals
                '\'' => {
                    self.lex_char();
                    let text = &self.source[start..self.pos];
                    tokens.push(Token::new(SyntaxKind::CharLit, self.span(start, self.pos), text));
                    continue;
                }

                // Operators and punctuation
                '(' => { self.advance(); SyntaxKind::LParen }
                ')' => { self.advance(); SyntaxKind::RParen }
                '{' => { self.advance(); SyntaxKind::LBrace }
                '}' => { self.advance(); SyntaxKind::RBrace }
                '[' => { self.advance(); SyntaxKind::LBracket }
                ']' => { self.advance(); SyntaxKind::RBracket }
                ',' => { self.advance(); SyntaxKind::Comma }
                ';' => { self.advance(); SyntaxKind::Semicolon }
                '.' => self.lex_dot(),
                ':' => self.lex_colon(),
                '+' => self.lex_plus(),
                '-' => self.lex_minus(),
                '*' => self.lex_star(),
                '/' => self.lex_slash(),
                '%' => { self.advance(); SyntaxKind::Percent }
                '=' => self.lex_eq(),
                '!' => self.lex_bang(),
                '<' => self.lex_lt(),
                '>' => self.lex_gt(),
                '&' => self.lex_and(),
                '|' => self.lex_or(),
                '^' => { self.advance(); SyntaxKind::Caret }
                '@' => { self.advance(); SyntaxKind::At }
                '#' => { self.advance(); SyntaxKind::Hash }
                '$' => { self.advance(); SyntaxKind::Dollar }
                '~' => { self.advance(); SyntaxKind::Tilde }
                '?' => { self.advance(); SyntaxKind::Question }

                _ => {
                    self.advance();
                    self.diagnostics.push(GlyimDiagnostic::lex_error(
                        self.span(start, self.pos),
                        format!("unexpected character: '{}'", ch),
                    ));
                    SyntaxKind::Error
                }
            };

            let text = &self.source[start..self.pos];
            tokens.push(Token::new(kind, self.span(start, self.pos), text));
        }

        LexResult {
            tokens,
            diagnostics: self.diagnostics,
        }
    }

    // ── Trivia ──

    fn lex_trivia(&mut self) -> bool {
        let start = self.pos;
        while let Some(ch) = self.peek() {
            match ch {
                ' ' | '\t' | '\n' | '\r' => {
                    self.advance();
                }
                '/' if self.peek_next() == Some('/') => {
                    self.advance(); // first /
                    self.advance(); // second /
                    while let Some(c) = self.peek() {
                        if c == '\n' { break; }
                        self.advance();
                    }
                }
                '/' if self.peek_next() == Some('*') => {
                    self.advance(); // /
                    self.advance(); // *
                    let mut depth = 1u32;
                    while depth > 0 {
                        match self.peek() {
                            Some('/') if self.peek_next() == Some('*') => {
                                self.advance(); self.advance(); depth += 1;
                            }
                            Some('*') if self.peek_next() == Some('/') => {
                                self.advance(); self.advance(); depth -= 1;
                            }
                            Some(_) => { self.advance(); }
                            None => break,
                        }
                    }
                }
                _ => break,
            }
        }
        self.pos > start
    }

    // ── Identifiers & keywords ──

    fn lex_ident_or_keyword(&mut self) {
        while let Some(ch) = self.peek() {
            if ch.is_alphanumeric() || ch == '_' {
                self.advance();
            } else {
                break;
            }
        }
    }

    // ── Numeric literals ──

    fn lex_number(&mut self) {
        while let Some(ch) = self.peek() {
            if ch.is_ascii_digit() || ch == '_' {
                self.advance();
            } else if ch == '.' && self.peek_next().map_or(false, |n| n.is_ascii_digit()) {
                self.advance(); // consume the dot
                while let Some(c) = self.peek() {
                    if c.is_ascii_digit() || c == '_' {
                        self.advance();
                    } else {
                        break;
                    }
                }
            } else {
                break;
            }
        }
        // Consume type suffix (i32, u8, f64, etc.)
        while let Some(ch) = self.peek() {
            if ch.is_alphanumeric() || ch == '_' {
                self.advance();
            } else {
                break;
            }
        }
    }

    // ── String literals ──

    fn lex_string(&mut self) {
        self.advance(); // opening "
        while let Some(ch) = self.peek() {
            match ch {
                '"' => { self.advance(); break; }
                '\\' => { self.advance(); self.advance(); }
                _ => { self.advance(); }
            }
        }
    }

    // ── Char literals ──

    fn lex_char(&mut self) {
        self.advance(); // opening '
        while let Some(ch) = self.peek() {
            match ch {
                '\'' => { self.advance(); break; }
                '\\' => { self.advance(); self.advance(); }
                _ => { self.advance(); }
            }
        }
    }

    // ── Compound operators ──

    fn lex_dot(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('.') {
            self.advance();
            if self.peek() == Some('=') {
                self.advance();
                SyntaxKind::DotDotEq
            } else {
                SyntaxKind::DotDot
            }
        } else {
            SyntaxKind::Dot
        }
    }

    fn lex_colon(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some(':') {
            self.advance();
            SyntaxKind::ColonColon
        } else {
            SyntaxKind::Colon
        }
    }

    fn lex_plus(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::PlusEq
        } else {
            SyntaxKind::Plus
        }
    }

    fn lex_minus(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('>') {
            self.advance();
            SyntaxKind::Arrow
        } else if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::MinusEq
        } else {
            SyntaxKind::Minus
        }
    }

    fn lex_star(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::StarEq
        } else {
            SyntaxKind::Star
        }
    }

    fn lex_slash(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::SlashEq
        } else {
            SyntaxKind::Slash
        }
    }

    fn lex_eq(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::EqEq
        } else if self.peek() == Some('>') {
            self.advance();
            SyntaxKind::FatArrow
        } else {
            SyntaxKind::Eq
        }
    }

    fn lex_bang(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::BangEq
        } else {
            SyntaxKind::Bang
        }
    }

    fn lex_lt(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::LtEq
        } else if self.peek() == Some('<') {
            self.advance();
            SyntaxKind::Shl
        } else {
            SyntaxKind::Lt
        }
    }

    fn lex_gt(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('=') {
            self.advance();
            SyntaxKind::GtEq
        } else if self.peek() == Some('>') {
            self.advance();
            SyntaxKind::Shr
        } else {
            SyntaxKind::Gt
        }
    }

    fn lex_and(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('&') {
            self.advance();
            SyntaxKind::AndAnd
        } else {
            SyntaxKind::And
        }
    }

    fn lex_or(&mut self) -> SyntaxKind {
        self.advance();
        if self.peek() == Some('|') {
            self.advance();
            SyntaxKind::OrOr
        } else {
            SyntaxKind::Or
        }
    }
}

// ── ParseResult ──

#[derive(Clone, Debug)]
pub struct ParseResult {

// ── Parse entry points ──

/// Parse a source string into a `SyntaxNode` tree.
///
/// This function is the explicit bridge from VFS to the CST.
/// It lexes the source, applies the Rowan parser, and returns
/// the root syntax node along with any diagnostics.
#[tracing::instrument(skip(source))]
pub fn parse_to_syntax(source: &str, file_id: FileId) -> ParseResult {
    let lex_result = glyim_lex::lex(source, file_id);
    let mut parser = Parser::new(&lex_result.tokens);

    parser.parse_source_file();

    let (green_node, diagnostics) = parser.finish();
    let root = SyntaxNode::new_root(green_node.clone());

    let mut all_diagnostics = lex_result.diagnostics;
    all_diagnostics.extend(diagnostics);

    ParseResult {
        green_node,
        diagnostics: all_diagnostics,
        root,
    }
}

// ── Parser ──

/// The incremental Rowan-based parser.
///
/// Uses `rowan::GreenNodeBuilder` to construct the tree incrementally.
/// Event-based: `start_node`, `finish_node`, `token`.
struct Parser<'a> {
    tokens: &'a [Token],
    pos: usize,
    builder: rowan::GreenNodeBuilder<'static>,
    diagnostics: Vec<GlyimDiagnostic>,
}

impl<'a> Parser<'a> {
    fn new(tokens: &'a [Token]) -> Self {
        Self {
            tokens,
            pos: 0,
            builder: rowan::GreenNodeBuilder::new(),
            diagnostics: Vec::new(),
        }
    }

    // ── Token access ──

    fn current(&self) -> Option<&Token> {
        self.tokens.get(self.pos)
    }

    fn current_kind(&self) -> SyntaxKind {
        self.current().map_or(SyntaxKind::Error, |t| t.kind)
    }

    fn lookahead(&self, offset: usize) -> Option<&Token> {
        self.tokens.get(self.pos + offset)
    }

    fn bump(&mut self) {
        if let Some(token) = self.current() {
            let kind = glyim_syntax::GlyimLang::kind_to_raw(token.kind);
            self.builder.token(
                rowan::SyntaxKind(kind),
                token.text.as_str(),
            );
            self.pos += 1;
        }
    }

    fn expect(&mut self, expected: SyntaxKind) {
        if self.current_kind() == expected {
            self.bump();
        } else {
            self.error(format!(
                "expected {:?}, found {:?}",
                expected,
                self.current_kind()
            ));
        }
    }

    fn error(&mut self, message: impl Into<String>) {
        let span = self.current()
            .map(|t| t.span)
            .unwrap_or(glyim_span::Span::DUMMY);
        self.diagnostics.push(GlyimDiagnostic::parse_error(span, message));
    }

    // ── Node construction ──

    fn start_node(&mut self, kind: SyntaxKind) {
        let raw = glyim_syntax::GlyimLang::kind_to_raw(kind);
        self.builder.start_node(rowan::SyntaxKind(raw));
    }

    fn finish_node(&mut self) {
        self.builder.finish_node();
    }

    // ── Parsing ──

    fn parse_source_file(&mut self) {
        self.start_node(SyntaxKind::SourceFile);
        while self.current().is_some() {
            self.parse_item();
        }
        self.finish_node();
    }

    fn parse_item(&mut self) {
        match self.current_kind() {
            SyntaxKind::KwFn => self.parse_fn_def(),
            SyntaxKind::KwStruct => self.parse_struct_def(),
            SyntaxKind::KwEnum => self.parse_enum_def(),
            SyntaxKind::KwTrait => self.parse_trait_def(),
            SyntaxKind::KwImpl => self.parse_impl_def(),
            SyntaxKind::KwType => self.parse_type_alias(),
            SyntaxKind::KwConst => self.parse_const_def(),
            SyntaxKind::KwStatic => self.parse_static_def(),
            SyntaxKind::KwMod => self.parse_mod(),
            SyntaxKind::KwUse => self.parse_use(),
            SyntaxKind::KwExtern => self.parse_extern(),
            _ => {
                self.error("expected item");
                self.bump();
            }
        }
    }

    fn parse_fn_def(&mut self) {
        self.start_node(SyntaxKind::FnDef);
        self.expect(SyntaxKind::KwFn);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::Lt {
            self.parse_type_param_list();
        }
        self.expect(SyntaxKind::LParen);
        self.start_node(SyntaxKind::ParamList);
        while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
            self.parse_param();
            if self.current_kind() == SyntaxKind::Comma {
                self.bump();
            }
        }
        self.finish_node(); // ParamList
        self.expect(SyntaxKind::RParen);
        if self.current_kind() == SyntaxKind::Arrow {
            self.bump();
            self.parse_type();
        }
        if self.current_kind() == SyntaxKind::LBrace {
            self.parse_block();
        }
        self.finish_node(); // FnDef
    }

    fn parse_struct_def(&mut self) {
        self.start_node(SyntaxKind::StructDef);
        self.expect(SyntaxKind::KwStruct);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::Lt {
            self.parse_type_param_list();
        }
        if self.current_kind() == SyntaxKind::LBrace {
            self.bump();
            while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
                self.bump();
            }
            self.expect(SyntaxKind::RBrace);
        } else if self.current_kind() == SyntaxKind::LParen {
            self.bump();
            while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
                self.bump();
            }
            self.expect(SyntaxKind::RParen);
            self.expect(SyntaxKind::Semicolon);
        } else {
            self.expect(SyntaxKind::Semicolon);
        }
        self.finish_node(); // StructDef
    }

    fn parse_enum_def(&mut self) {
        self.start_node(SyntaxKind::EnumDef);
        self.expect(SyntaxKind::KwEnum);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::Lt {
            self.parse_type_param_list();
        }
        self.expect(SyntaxKind::LBrace);
        while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
            self.bump();
        }
        self.expect(SyntaxKind::RBrace);
        self.finish_node(); // EnumDef
    }

    fn parse_trait_def(&mut self) {
        self.start_node(SyntaxKind::TraitDef);
        self.expect(SyntaxKind::KwTrait);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::LBrace {
            self.parse_block();
        }
        self.finish_node(); // TraitDef
    }

    fn parse_impl_def(&mut self) {
        self.start_node(SyntaxKind::ImplDef);
        self.expect(SyntaxKind::KwImpl);
        if self.current_kind() == SyntaxKind::Lt {
            self.parse_type_param_list();
        }
        self.parse_type(); // self type
        if self.current_kind() == SyntaxKind::KwFor {
            self.bump();
            self.parse_type(); // trait type
        }
        if self.current_kind() == SyntaxKind::LBrace {
            self.parse_block();
        }
        self.finish_node(); // ImplDef
    }

    fn parse_type_alias(&mut self) {
        self.start_node(SyntaxKind::TypeAlias);
        self.expect(SyntaxKind::KwType);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::Eq {
            self.bump();
            self.parse_type();
        }
        self.expect(SyntaxKind::Semicolon);
        self.finish_node(); // TypeAlias
    }

    fn parse_const_def(&mut self) {
        self.start_node(SyntaxKind::ConstDef);
        self.expect(SyntaxKind::KwConst);
        self.expect(SyntaxKind::Ident);
        self.expect(SyntaxKind::Colon);
        self.parse_type();
        if self.current_kind() == SyntaxKind::Eq {
            self.bump();
            self.parse_expr();
        }
        self.expect(SyntaxKind::Semicolon);
        self.finish_node(); // ConstDef
    }

    fn parse_static_def(&mut self) {
        self.start_node(SyntaxKind::StaticDef);
        self.expect(SyntaxKind::KwStatic);
        self.expect(SyntaxKind::Ident);
        self.expect(SyntaxKind::Colon);
        self.parse_type();
        if self.current_kind() == SyntaxKind::Eq {
            self.bump();
            self.parse_expr();
        }
        self.expect(SyntaxKind::Semicolon);
        self.finish_node(); // StaticDef
    }

    fn parse_mod(&mut self) {
        self.start_node(SyntaxKind::Module);
        self.expect(SyntaxKind::KwMod);
        self.expect(SyntaxKind::Ident);
        if self.current_kind() == SyntaxKind::LBrace {
            self.parse_block();
        } else {
            self.expect(SyntaxKind::Semicolon);
        }
        self.finish_node(); // Module
    }

    fn parse_use(&mut self) {
        self.start_node(SyntaxKind::UseDecl);
        self.expect(SyntaxKind::KwUse);
        self.parse_path();
        self.expect(SyntaxKind::Semicolon);
        self.finish_node(); // UseDecl
    }

    fn parse_extern(&mut self) {
        self.start_node(SyntaxKind::ExternBlock);
        self.expect(SyntaxKind::KwExtern);
        if self.current_kind() == SyntaxKind::StringLit {
            self.bump();
        }
        self.expect(SyntaxKind::LBrace);
        while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
            self.parse_item();
        }
        self.expect(SyntaxKind::RBrace);
        self.finish_node(); // ExternBlock
    }

    fn parse_param(&mut self) {
        self.start_node(SyntaxKind::Param);
        self.expect(SyntaxKind::Ident);
        self.expect(SyntaxKind::Colon);
        self.parse_type();
        self.finish_node(); // Param
    }

    fn parse_type_param_list(&mut self) {
        self.start_node(SyntaxKind::TypeParamList);
        self.expect(SyntaxKind::Lt);
        while self.current_kind() != SyntaxKind::Gt && self.current().is_some() {
            self.start_node(SyntaxKind::TypeParam);
            self.expect(SyntaxKind::Ident);
            self.finish_node(); // TypeParam
            if self.current_kind() == SyntaxKind::Comma {
                self.bump();
            }
        }
        self.expect(SyntaxKind::Gt);
        self.finish_node(); // TypeParamList
    }

    fn parse_block(&mut self) {
        self.start_node(SyntaxKind::Block);
        self.expect(SyntaxKind::LBrace);
        while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
            self.parse_stmt();
        }
        self.expect(SyntaxKind::RBrace);
        self.finish_node(); // Block
    }

    fn parse_stmt(&mut self) {
        match self.current_kind() {
            SyntaxKind::KwLet => {
                self.start_node(SyntaxKind::LetStmt);
                self.expect(SyntaxKind::KwLet);
                self.parse_pat();
                if self.current_kind() == SyntaxKind::Colon {
                    self.bump();
                    self.parse_type();
                }
                if self.current_kind() == SyntaxKind::Eq {
                    self.bump();
                    self.parse_expr();
                }
                self.expect(SyntaxKind::Semicolon);
                self.finish_node();
            }
            _ => {
                self.start_node(SyntaxKind::ExprStmt);
                self.parse_expr();
                if self.current_kind() == SyntaxKind::Semicolon {
                    self.bump();
                }
                self.finish_node();
            }
        }
    }

    fn parse_expr(&mut self) {
        self.parse_assignment_expr();
    }

    fn parse_assignment_expr(&mut self) {
        self.parse_or_expr();
        if self.current_kind() == SyntaxKind::Eq {
            self.start_node(SyntaxKind::AssignExpr);
            self.bump();
            self.parse_expr();
            self.finish_node();
        }
    }

    fn parse_or_expr(&mut self) {
        self.parse_and_expr();
        while self.current_kind() == SyntaxKind::OrOr {
            self.start_node(SyntaxKind::BinaryExpr);
            self.bump();
            self.parse_and_expr();
            self.finish_node();
        }
    }

    fn parse_and_expr(&mut self) {
        self.parse_comparison_expr();
        while self.current_kind() == SyntaxKind::AndAnd {
            self.start_node(SyntaxKind::BinaryExpr);
            self.bump();
            self.parse_comparison_expr();
            self.finish_node();
        }
    }

    fn parse_comparison_expr(&mut self) {
        self.parse_additive_expr();
        if matches!(self.current_kind(),
            SyntaxKind::EqEq | SyntaxKind::BangEq |
            SyntaxKind::Lt | SyntaxKind::Gt |
            SyntaxKind::LtEq | SyntaxKind::GtEq
        ) {
            self.start_node(SyntaxKind::BinaryExpr);
            self.bump();
            self.parse_additive_expr();
            self.finish_node();
        }
    }

    fn parse_additive_expr(&mut self) {
        self.parse_multiplicative_expr();
        while matches!(self.current_kind(), SyntaxKind::Plus | SyntaxKind::Minus) {
            self.start_node(SyntaxKind::BinaryExpr);
            self.bump();
            self.parse_multiplicative_expr();
            self.finish_node();
        }
    }

    fn parse_multiplicative_expr(&mut self) {
        self.parse_unary_expr();
        while matches!(self.current_kind(), SyntaxKind::Star | SyntaxKind::Slash | SyntaxKind::Percent) {
            self.start_node(SyntaxKind::BinaryExpr);
            self.bump();
            self.parse_unary_expr();
            self.finish_node();
        }
    }

    fn parse_unary_expr(&mut self) {
        if matches!(self.current_kind(), SyntaxKind::Bang | SyntaxKind::Minus | SyntaxKind::Star | SyntaxKind::And) {
            self.start_node(SyntaxKind::UnaryExpr);
            self.bump();
            self.parse_unary_expr();
            self.finish_node();
        } else {
            self.parse_postfix_expr();
        }
    }

    fn parse_postfix_expr(&mut self) {
        self.parse_primary_expr();
        loop {
            match self.current_kind() {
                SyntaxKind::LParen => {
                    self.start_node(SyntaxKind::CallExpr);
                    self.bump();
                    while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
                        self.parse_expr();
                        if self.current_kind() == SyntaxKind::Comma { self.bump(); }
                    }
                    self.expect(SyntaxKind::RParen);
                    self.finish_node();
                }
                SyntaxKind::Dot => {
                    self.start_node(SyntaxKind::FieldExpr);
                    self.bump();
                    self.expect(SyntaxKind::Ident);
                    if self.current_kind() == SyntaxKind::LParen {
                        self.start_node(SyntaxKind::MethodCallExpr);
                        self.bump();
                        while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
                            self.parse_expr();
                            if self.current_kind() == SyntaxKind::Comma { self.bump(); }
                        }
                        self.expect(SyntaxKind::RParen);
                        self.finish_node();
                    }
                    self.finish_node();
                }
                SyntaxKind::LBracket => {
                    self.start_node(SyntaxKind::IndexExpr);
                    self.bump();
                    self.parse_expr();
                    self.expect(SyntaxKind::RBracket);
                    self.finish_node();
                }
                _ => break,
            }
        }
    }

    fn parse_primary_expr(&mut self) {
        match self.current_kind() {
            SyntaxKind::Ident => self.parse_path_expr(),
            SyntaxKind::IntLit | SyntaxKind::FloatLit | SyntaxKind::StringLit
            | SyntaxKind::CharLit | SyntaxKind::KwTrue | SyntaxKind::KwFalse => {
                self.start_node(SyntaxKind::LitExpr);
                self.bump();
                self.finish_node();
            }
            SyntaxKind::LParen => {
                self.bump();
                if self.current_kind() == SyntaxKind::RParen {
                    self.bump();
                } else {
                    self.parse_expr();
                    self.expect(SyntaxKind::RParen);
                }
            }
            SyntaxKind::LBrace => self.parse_block(),
            SyntaxKind::KwIf => self.parse_if_expr(),
            SyntaxKind::KwWhile => self.parse_while_expr(),
            SyntaxKind::KwFor => self.parse_for_expr(),
            SyntaxKind::KwMatch => self.parse_match_expr(),
            SyntaxKind::KwReturn => {
                self.bump();
                if !matches!(self.current_kind(), SyntaxKind::Semicolon | SyntaxKind::RBrace) {
                    self.parse_expr();
                }
            }
            SyntaxKind::KwBreak => { self.bump(); }
            SyntaxKind::KwContinue => { self.bump(); }
            _ => {
                self.error("expected expression");
                self.bump();
            }
        }
    }

    fn parse_path_expr(&mut self) {
        self.start_node(SyntaxKind::PathExpr);
        self.parse_path();
        if self.current_kind() == SyntaxKind::LBrace {
            self.start_node(SyntaxKind::StructExpr);
            self.bump();
            while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
                self.expect(SyntaxKind::Ident);
                self.expect(SyntaxKind::Colon);
                self.parse_expr();
                if self.current_kind() == SyntaxKind::Comma { self.bump(); }
            }
            self.expect(SyntaxKind::RBrace);
            self.finish_node();
        }
        self.finish_node(); // PathExpr
    }

    fn parse_path(&mut self) {
        self.start_node(SyntaxKind::UsePath);
        if self.current_kind() == SyntaxKind::KwSelf {
            self.bump();
        } else if self.current_kind() == SyntaxKind::KwSuper {
            self.bump();
        } else if self.current_kind() == SyntaxKind::KwCrate {
            self.bump();
        } else {
            self.expect(SyntaxKind::Ident);
        }
        while self.current_kind() == SyntaxKind::ColonColon {
            self.bump();
            self.expect(SyntaxKind::Ident);
        }
        self.finish_node(); // UsePath
    }

    fn parse_if_expr(&mut self) {
        self.start_node(SyntaxKind::IfExpr);
        self.expect(SyntaxKind::KwIf);
        self.parse_expr();
        self.parse_block();
        if self.current_kind() == SyntaxKind::KwElse {
            self.bump();
            if self.current_kind() == SyntaxKind::KwIf {
                self.parse_if_expr();
            } else {
                self.parse_block();
            }
        }
        self.finish_node(); // IfExpr
    }

    fn parse_while_expr(&mut self) {
        self.start_node(SyntaxKind::WhileExpr);
        self.expect(SyntaxKind::KwWhile);
        self.parse_expr();
        self.parse_block();
        self.finish_node(); // WhileExpr
    }

    fn parse_for_expr(&mut self) {
        self.start_node(SyntaxKind::ForExpr);
        self.expect(SyntaxKind::KwFor);
        self.parse_pat();
        self.expect(SyntaxKind::KwIn);
        self.parse_expr();
        self.parse_block();
        self.finish_node(); // ForExpr
    }

    fn parse_match_expr(&mut self) {
        self.start_node(SyntaxKind::MatchExpr);
        self.expect(SyntaxKind::KwMatch);
        self.parse_expr();
        self.expect(SyntaxKind::LBrace);
        self.start_node(SyntaxKind::MatchArmList);
        while self.current_kind() != SyntaxKind::RBrace && self.current().is_some() {
            self.start_node(SyntaxKind::MatchArm);
            self.parse_pat();
            if self.current_kind() == SyntaxKind::KwIf {
                self.bump();
                self.parse_expr();
            }
            self.expect(SyntaxKind::FatArrow);
            self.parse_expr();
            if self.current_kind() == SyntaxKind::Comma { self.bump(); }
            self.finish_node(); // MatchArm
        }
        self.finish_node(); // MatchArmList
        self.expect(SyntaxKind::RBrace);
        self.finish_node(); // MatchExpr
    }

    fn parse_pat(&mut self) {
        match self.current_kind() {
            SyntaxKind::Underscore => {
                self.start_node(SyntaxKind::PatWild);
                self.bump();
                self.finish_node();
            }
            SyntaxKind::Ident => {
                self.start_node(SyntaxKind::PatIdent);
                self.bump();
                self.finish_node();
            }
            _ => {
                self.error("expected pattern");
                self.bump();
            }
        }
    }

    fn parse_type(&mut self) {
        match self.current_kind() {
            SyntaxKind::Ident => {
                self.start_node(SyntaxKind::PathType);
                self.parse_path();
                self.finish_node();
            }
            SyntaxKind::And => {
                self.start_node(SyntaxKind::RefType);
                self.bump();
                if self.current_kind() == SyntaxKind::KwMut { self.bump(); }
                self.parse_type();
                self.finish_node();
            }
            SyntaxKind::LBracket => {
                self.bump();
                self.parse_type();
                if self.current_kind() == SyntaxKind::Semicolon {
                    self.start_node(SyntaxKind::ArrayType);
                    self.bump();
                    self.parse_expr();
                    self.expect(SyntaxKind::RBracket);
                    self.finish_node();
                } else {
                    self.start_node(SyntaxKind::SliceType);
                    self.expect(SyntaxKind::RBracket);
                    self.finish_node();
                }
            }
            SyntaxKind::LParen => {
                self.start_node(SyntaxKind::TupleType);
                self.bump();
                while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
                    self.parse_type();
                    if self.current_kind() == SyntaxKind::Comma { self.bump(); }
                }
                self.expect(SyntaxKind::RParen);
                self.finish_node();
            }
            SyntaxKind::KwFn => {
                self.start_node(SyntaxKind::FnType);
                self.bump();
                self.expect(SyntaxKind::LParen);
                while self.current_kind() != SyntaxKind::RParen && self.current().is_some() {
                    self.parse_type();
                    if self.current_kind() == SyntaxKind::Comma { self.bump(); }
                }
                self.expect(SyntaxKind::RParen);
                if self.current_kind() == SyntaxKind::Arrow {
                    self.bump();
                    self.parse_type();
                }
                self.finish_node();
            }
            SyntaxKind::Bang => {
                self.start_node(SyntaxKind::NeverType);
                self.bump();
                self.finish_node();
            }
            _ => {
                self.error("expected type");
                self.bump();
            }
        }
    }

    // ── Finish ──

    fn finish(self) -> (GreenNode, Vec<GlyimDiagnostic>) {
        (self.builder.finish(), self.diagnostics)
    }
}

// crates/glyim-def-map/src/lib.rs
//! Module graph and lightweight definition map.
//!
//! Builds the module tree and collects item declarations from the CST.
//! No type information is resolved here — only names and visibility.

use glyim_arena::IndexVec;
use glyim_primitives::Visibility;
use glyim_interner::Name;
use glyim_span::{Span, FileId};
use glyim_syntax::SyntaxNode;
use glyim_diag::GlyimDiagnostic;

glyim_arena::define_idx!(ModuleId);
glyim_arena::define_idx!(DefId);

// ── CrateDefMap ──

#[derive(Clone, Debug)]
pub struct CrateDefMap {
    pub root: ModuleId,
    pub modules: IndexVec<ModuleId, ModuleData>,
    pub defs: IndexVec<DefId, DefData>,
    pub krate: CrateId,
}

// ── CrateId ──

#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash)]
pub struct CrateId(pub u32);

// ── ModuleData ──

#[derive(Clone, Debug)]
pub struct ModuleData {
    pub parent: Option<ModuleId>,
    pub children: Vec<(Name, ModuleId)>,
    pub scope: ItemScope,
    pub origin: ModuleOrigin,
    pub span: Span,
}

impl ModuleData {
    pub fn resolve(&self, name: Name) -> Option<(DefId, Visibility)> {
        self.scope.resolve(name)
    }
}

#[derive(Clone, Debug)]
pub enum ModuleOrigin {
    File { file_id: FileId },
    Inline { span: Span },
    CrateRoot,
}

// ── ItemScope ──

#[derive(Clone, Debug, Default)]
pub struct ItemScope {
    pub types: Vec<(Name, DefId, Visibility, Span)>,
    pub values: Vec<(Name, DefId, Visibility, Span)>,
    pub macros: Vec<(Name, DefId, Visibility, Span)>,
}

impl ItemScope {
    pub fn resolve(&self, name: Name) -> Option<(DefId, Visibility)> {
        self.types
            .iter()
            .chain(self.values.iter())
            .find(|(n, _, _, _)| *n == name)
            .map(|(_, id, vis, _)| (*id, *vis))
    }

    pub fn declare(
        &mut self,
        name: Name,
        id: DefId,
        vis: Visibility,
        span: Span,
        ns: Namespace,
    ) {
        let entry = (name, id, vis, span);
        match ns {
            Namespace::Types => self.types.push(entry),

        parent: None,
        children: Vec::new(),
        scope: ItemScope::default(),
        origin: ModuleOrigin::CrateRoot,
        span: glyim_span::Span::DUMMY,
    });

    // Walk the CST and collect definitions
    for child in root.children() {
        collect_item(&child, root_module, &mut modules, &mut defs, &mut diagnostics);
    }

    let def_map = CrateDefMap {
        root: root_module,
        modules,
        defs,
        krate,
    };

}

pub struct HygieneCtx {
    expansions: Vec<ExpnData>,
    next_expn_id: u32,
    syntax_contexts: Vec<SyntaxContextData>,
    next_syntax_context: u32,
    /// [B1] Hold the key securely inside the hygiene crate.

#[derive(Clone, Debug)]
struct SyntaxContextData {
    outer_expn: ExpnId,
    outer_transparency: Transparency,
    parent: SyntaxContext,
    /// Computed: this context "adjusts" through the given scope.
    adjusted: Option<SyntaxContext>,
}

impl HygieneCtx {
    pub fn new() -> Self {
        // SAFETY: This constructor is inside `glyim-hygiene`,
        // upholding the invariant that no other crate calls this.
        let key = unsafe { HygieneKey::new_unchecked() };

        Self {
            expansions: vec![ExpnData {
                expn_id: ExpnId::ROOT,
                parent: ExpnId::ROOT,
                kind: ExpnKind::Root,
                call_site: Span::DUMMY,
                def_site: Span::DUMMY,
                transparency: Transparency::Opaque,
            }],
            next_expn_id: 1,
            syntax_contexts: Vec::new(),
            next_syntax_context: 1, // 0 is ROOT
        }
    }

    /// [B1] Construct an `ExpnId` safely using the HygieneKey.
    #[tracing::instrument(skip(self))]
    pub fn push_expansion(&mut self, mut data: ExpnData) -> ExpnId {
        let raw_id = self.next_expn_id;
        self.next_expn_id += 1;

        let id = ExpnId::from_raw(self.key, raw_id);
        data.expn_id = id;
        self.expansions.push(data);
        id
    }

    /// Apply a hygiene mark to a span.
    #[tracing::instrument(skip(self))]
    pub fn apply_mark(&mut self, span: Span, mark: Mark) -> Span {
        let new_ctx = SyntaxContext::from_raw(self.key, self.next_syntax_context);
        self.next_syntax_context += 1;

        self.syntax_contexts.push(SyntaxContextData {
            outer_expn: mark.expn_id,
            outer_transparency: mark.transparency,
            parent: span.ctx,
            adjusted: None,
        });

        Span::new(span.file, span.lo, span.hi, new_ctx)
    }

    /// Remove the outermost mark from a span.
    #[tracing::instrument(skip(self))]
    pub fn remove_mark(&self, span: Span) -> (Span, Option<Mark>) {
        if span.ctx.is_root() {
            return (span, None);
        }

        let idx = span.ctx.to_raw() as usize - 1;
        if let Some(ctx_data) = self.syntax_contexts.get(idx) {
            let mark = Mark {
                expn_id: ctx_data.outer_expn,
                transparency: ctx_data.outer_transparency,
            };
            let new_span = Span::new(span.file, span.lo, span.hi, ctx_data.parent);
            (new_span, Some(mark))
        } else {
            (span, None)
        }
    }

    /// Look up expansion data.
    pub fn expn_data(&self, id: ExpnId) -> Option<&ExpnData> {
        self.expansions.get(id.to_raw() as usize)
    }

    /// Adjust a span's context for a given scope.
    pub fn adjust(&mut self, span: Span, scope_ctx: SyntaxContext) -> Span {
        // Walk the context chain, removing marks that are transparent
        // relative to the scope. Stub: return as-is for v0.1.0.
        let _ = scope_ctx;
        span
    }
}

impl Default for HygieneCtx {
    fn default() -> Self {
        Self::new()
    }
}

//!
//! # The Spine
//!
//! `TyCtx` is the compiler's spine — the single source of truth for
//! all type data after type checking. It is:
//! - **Frozen** after construction: all methods take `&self`.
//! - **`Send + Sync`**: safe to share across threads.
//! - **Owned by `glyim-db`**: the driver constructs it via
//!   `TyCtxMut::freeze()` and stores it for the rest of compilation.

//!

//! allocating types, interning substitutions, etc. After typeck

            goals: indexmap::IndexSet::new(),
            goals_list: indexmap::IndexSet::new(),
            substitutions: indexmap::IndexSet::new(),
            program_clauses: indexmap::IndexSet::new(),
            program_clause: indexmap::IndexSet::new(),

    }

/// The custom Chalk interner for Glyim.
///
/// [C2] Holds a raw pointer to `ChalkArenas`. This is `Copy`,
/// satisfying Chalk's `Interner: Copy` bound. The pointer is only
/// constructed inside `typeck_crate` where `&mut TyCtxMut` is held,
/// guaranteeing exclusive access and pointer validity.
/// No global state. `ChalkArenas` is dropped on `freeze()`.
#[derive(Copy, Clone, PartialEq, Eq, Debug, Hash, PartialOrd, Ord)]
pub struct GlyimChalkInterner {
    arenas: *mut ChalkArenas,
}

// SAFETY: The pointer is only used during `typeck_crate` where
// `&mut TyCtxMut` guarantees exclusive, thread-local access.
// It does not escape to `TyCtx`.
unsafe impl Send for GlyimChalkInterner {}
unsafe impl Sync for GlyimChalkInterner {}

impl GlyimChalkInterner {
    /// Construct the interner. Only called by `TyCtxMut`.
    ///
    /// # Safety
    ///
    /// `arenas` must be valid for the duration of the typeck pass
    /// and must not be shared across threads.
    pub unsafe fn new(arenas: *mut ChalkArenas) -> Self {
        Self { arenas }
    }

    /// Access the arenas. Only safe during typeck.
    ///
    /// # Safety
    ///
    /// Must only be called while `TyCtxMut` is held exclusively.
    pub unsafe fn arenas(&self) -> &mut ChalkArenas {
        &mut *self.arenas
    }
}

// ══════════════════════════════════════════════════════════════════
// Part 4: TyCtxMut — Mutable, Chalk execution happens here
// ══════════════════════════════════════════════════════════════════

/// Mutable type context. Used **only** during `glyim-typeck`.
///
/// [C1] This is NOT `Send + Sync` — it is confined to a single
/// typeck thread/task. After typeck completes, call `freeze()` to
/// obtain an immutable `TyCtx` that IS `Send + Sync`.
pub struct TyCtxMut {
    /// Type interner arena.
    types: IndexVec<Ty, TyKind>,
    type_flags: Vec<TypeFlags>,

    /// [P1] Substitution interner arena.
    /// Uses `IndexSet` for zero-alloc lookups on existing entries.
    substitution_data: indexmap::IndexSet<Vec<GenericArg>>,

    /// Region interner arena.
    regions: IndexVec<RegionVid, Region>,

    /// String interner resolver (read-only).
    resolver: glyim_interner::Resolver,

    /// [C1/C2] Owned Chalk scratchpad. Dropped on freeze.
    chalk_arenas: ChalkArenas,
}

impl TyCtxMut {
    pub fn new(resolver: glyim_interner::Resolver) -> Self {
        let mut ctx = Self {
            types: IndexVec::new(),
            type_flags: Vec::new(),
            substitution_data: indexmap::IndexSet::new(),
            regions: IndexVec::new(),
            resolver,
            chalk_arenas: ChalkArenas::new(),
        };
        // Pre-intern sentinel types at well-known indices.
        ctx.alloc_ty(TyKind::Error);  // Ty(0)
        ctx.alloc_ty(TyKind::Never);  // Ty(1)
        ctx.alloc_ty(TyKind::Unit);   // Ty(2)
        ctx.alloc_ty(TyKind::Bool);   // Ty(3)
        ctx
    }

    // ── Type allocation ──

    #[tracing::instrument(skip(self))]
    pub fn alloc_ty(&mut self, kind: TyKind) -> Ty {
        let flags = compute_flags(&kind);
        let idx = self.types.push(kind);
        self.type_flags.push(flags);
        Ty::from_raw(idx.to_raw())
    }

    pub fn ty_kind(&self, ty: Ty) -> &TyKind {
        &self.types[ty]
    }

    pub fn ty_kind_mut(&mut self, ty: Ty) -> &mut TyKind {
        &mut self.types[ty]
    }

    pub fn ty_flags(&self, ty: Ty) -> TypeFlags {
        self.type_flags[ty.to_raw() as usize]
    }

    // ── [P1] Zero-alloc Substitution Interning ──

    /// Intern a substitution. `insert_full` borrows `args` as a slice,
    /// so no `Box` allocation occurs for existing entries.
    #[tracing::instrument(skip(self))]
    pub fn intern_substitution(&mut self, args: Vec<GenericArg>) -> Substitution {
        let len = args.len() as u16;
        let (index, _) = self.substitution_data.insert_full(args);
        Substitution::from_raw(index as u32, len)
    }

    /// Look up the arguments for an interned substitution.
    pub fn substitution_args(&self, sub: Substitution) -> &[GenericArg] {
        &self.substitution_data[sub.index() as usize]
    }

    // ── Convenience constructors ──

    pub fn mk_ty(&mut self, kind: TyKind) -> Ty {
        self.alloc_ty(kind)
    }

    pub fn mk_ref(&mut self, region: Region, ty: Ty, mutability: Mutability) -> Ty {
        self.mk_ty(TyKind::Ref(region, ty, mutability))
    }

    pub fn mk_raw_ptr(&mut self, ty: Ty, mutability: Mutability) -> Ty {
        self.mk_ty(TyKind::RawPtr(ty, mutability))
    }

    pub fn mk_slice(&mut self, ty: Ty) -> Ty {
        self.mk_ty(TyKind::Slice(ty))
    }

    pub fn mk_array(&mut self, ty: Ty, len: Const) -> Ty {
        self.mk_ty(TyKind::Array(ty, len))
    }

    pub fn mk_adt(&mut self, id: AdtId, substs: Substitution) -> Ty {
        self.mk_ty(TyKind::Adt(id, substs))
    }

    pub fn mk_fn_def(&mut self, id: FnDefId, substs: Substitution) -> Ty {
        self.mk_ty(TyKind::FnDef(id, substs))
    }

    pub fn mk_tuple(&mut self, tys: Vec<GenericArg>) -> Ty {
        let sub = self.intern_substitution(tys);
        self.mk_ty(TyKind::Tuple(sub))
    }

    pub fn mk_fn_ptr(&mut self, sig: FnSig) -> Ty {
        self.mk_ty(TyKind::FnPtr(sig))
    }

    pub fn mk_error(&self) -> Ty {
        Ty::from_raw(0)
    }

    pub fn mk_never(&self) -> Ty {
        Ty::from_raw(1)
    }

    pub fn mk_unit(&self) -> Ty {
        Ty::from_raw(2)
    }

    pub fn mk_bool(&self) -> Ty {
        Ty::from_raw(3)
    }

    // ── Region allocation ──

    pub fn new_region_var(&mut self) -> RegionVid {
        self.regions.push(Region::Error)
    }

    pub fn set_region(&mut self, vid: RegionVid, region: Region) {
        self.regions[vid] = region;
    }

    pub fn region(&self, vid: RegionVid) -> &Region {
        &self.regions[vid]
    }

    // ── String resolution ──

    pub fn resolver(&self) -> &glyim_interner::Resolver {
        &self.resolver
    }

    // ── Chalk access ──

    /// [C2] Construct the Chalk interner.
    ///
    /// # Safety
    ///
    /// Must not outlive `self`. Only used during `typeck_crate`.
    pub unsafe fn chalk_interner(&mut self) -> GlyimChalkInterner {
        GlyimChalkInterner::new(&mut self.chalk_arenas)
    }

    // ── [C1] Freeze: consume TyCtxMut, produce immutable TyCtx ──

    /// Freeze the mutable type context into an immutable one.
    ///
    /// [C1] Drops `ChalkArenas`, ensuring no mutation can happen later.
    /// The returned `TyCtx` is `Send + Sync` (auto-derived, no `unsafe`).
    pub fn freeze(self) -> TyCtx {
        TyCtx {
            types: self.types,
            type_flags: self.type_flags,
            // Convert IndexSet to Vec for simpler storage
            substitution_data: self.substitution_data.into_iter().collect(),
            regions: self.regions,
            resolver: self.resolver,
            // chalk_arenas is DROPPED here. No Chalk mutation possible.
        }
    }
}

fn compute_flags(kind: &TyKind) -> TypeFlags {
    let mut flags = TypeFlags::empty();
    match kind {
        TyKind::Infer(_) => flags |= TypeFlags::HAS_TY_INFER,
        TyKind::Param(_) => flags |= TypeFlags::HAS_TY_PARAM,
        TyKind::Error => flags |= TypeFlags::HAS_ERROR,
        TyKind::Ref(Region::Var(_), _, _) => {
            flags |= TypeFlags::HAS_RE_INFER;
        }
        TyKind::Ref(Region::EarlyBound(_), _, _) => {
            flags |= TypeFlags::HAS_RE_PARAM;
        }
        _ => {}
    }
    flags
}

/// Wrapper for debug-printing a type with its `TyCtx`.
///
/// Usage: `format!("{}", PrintTy(ty, ctx))`
///
/// Or use the `debug_ty!` macro for convenience.
pub struct PrintTy<'a>(pub Ty, pub &'a TyCtx);

impl<'a> std::fmt::Display for PrintTy<'a> {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self.1.ty_kind(self.0) {
            TyKind::Bool => write!(f, "bool"),
            TyKind::Never => write!(f, "!"),
            TyKind::Unit => write!(f, "()"),
            TyKind::Int(i) => write!(f, "{}", i.name()),
            TyKind::Uint(u) => write!(f, "{}", u.name()),
            TyKind::Float(fl) => write!(f, "{}", fl.name()),
            TyKind::Char => write!(f, "char"),
            TyKind::String => write!(f, "String"),
            TyKind::Error => write!(f, "<error>"),
            TyKind::Infer(InferVar::Ty(v)) => write!(f, "?{}", v.to_raw()),
            TyKind::Infer(InferVar::Int(v)) => write!(f, "?{}", v.to_raw()),
            TyKind::Infer(InferVar::Float(v)) => write!(f, "?{}", v.to_raw()),
            TyKind::Param(p) => write!(f, "{}", self.1.name_str(p.name)),
            TyKind::Ref(_, ty, Mutability::Mut) => {
                write!(f, "&mut {}", PrintTy(*ty, self.1))
            }
            TyKind::Ref(_, ty, Mutability::Not) => {
                write!(f, "&{}", PrintTy(*ty, self.1))
            }
            TyKind::RawPtr(ty, Mutability::Mut) => {
                write!(f, "*mut {}", PrintTy(*ty, self.1))
            }
            TyKind::RawPtr(ty, Mutability::Not) => {
                write!(f, "*const {}", PrintTy(*ty, self.1))
            }
            TyKind::Slice(ty) => write!(f, "[{}]", PrintTy(*ty, self.1)),
            TyKind::Adt(id, substs) => {
                write!(f, "Adt({}", id.0)?;
                if !substs.is_empty() {
                    write!(f, "<")?;
                    let args = self.1.substitution_args(*substs);
                    for (i, arg) in args.iter().enumerate() {
                        if i > 0 {
                            write!(f, ", ")?;
                        }
                        match arg {
                            GenericArg::Ty(t) => write!(f, "{}", PrintTy(*t, self.1))?,
                            GenericArg::Lifetime(_) => write!(f, "'_")?,
                            GenericArg::Const(_) => write!(f, "const")?,
                        }
                    }
                    write!(f, ">")?;
                }
                write!(f, ")")
            }
            TyKind::FnPtr(sig) => {
                write!(f, "fn(")?;
                let args = self.1.substitution_args(sig.inputs);
                for (i, arg) in args.iter().enumerate() {
                    if i > 0 {
                        write!(f, ", ")?;
                    }
                    if let GenericArg::Ty(t) = arg {
                        write!(f, "{}", PrintTy(*t, self.1))?;
                    }
                }
                write!(f, ") -> {}", PrintTy(sig.output, self.1))
            }
            TyKind::Tuple(subs) => {
                write!(f, "(")?;
                let args = self.1.substitution_args(*subs);
                for (i, arg) in args.iter().enumerate() {
                    if i > 0 {
                        write!(f, ", ")?;
                    }
                    if let GenericArg::Ty(t) = arg {
                        write!(f, "{}", PrintTy(*t, self.1))?;
                    }
                }
                write!(f, ")")
            }
            other => write!(f, "{:?}", other),
        }
    }
}

/// Convenience macro for debug-printing a type.
///
/// Usage: `debug_ty!(ctx, ty)` → `PrintTy(ty, ctx)`
#[macro_export]
macro_rules! debug_ty {
    ($ctx:expr, $ty:expr) => {
        $crate::PrintTy($ty, $ctx)
    };
}

                    TyKind::Ref(_, inner, _) => *inner,
                    TyKind::RawPtr(inner, _) => *inner,
                    _ => ctx.error_ty(),

            ProjectionElem::Field(_) => {
                // Stub: real implementation looks up field type
                ty

                    TyKind::Array(inner, _) => *inner,
                    TyKind::Slice(inner) => *inner,
                    _ => ctx.error_ty(),

//! Type layout computation — sizes, alignments, ABI details.
//!
//! Layouts are computed from `Ty` values using a `TyCtx`. The
//! `LayoutComputer` trait abstracts the computation, allowing
//! different backends to provide target-specific logic.

use glyim_arena::IndexVec;
use glyim_type::*;

// ── Primitives ──

#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash)]
pub struct Size(pub u64);

impl Size {
    pub const ZERO: Size = Size(0);

    pub fn bytes(b: u64) -> Self {
        Size(b)
    }

    pub fn bits(&self) -> u64 {
        self.0 * 8
    }

    /// Align the size up to the given alignment.
    pub fn align_to(&self, align: Align) -> Self {
        let mask = align.0 - 1;
        Size((self.0 + mask) & !mask)
    }

    pub fn is_aligned(&self, align: Align) -> bool {
        self.0 % align.0 == 0
    }
}

impl std::ops::Add for Size {
    type Output = Size;
    fn add(self, rhs: Size) -> Size {
        Size(self.0 + rhs.0)
    }
}

#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash)]
pub struct Align(pub u64); // Must be a power of 2

impl Align {
    pub const ONE: Align = Align(1);
    pub const EIGHT: Align = Align(8);

    pub fn from_bytes(bytes: u64) -> Self {
        debug_assert!(bytes.is_power_of_two());
        Align(bytes)
    }

    pub fn max(self, other: Self) -> Self {
        Align(self.0.max(other.0))
    }

    pub fn min(self, other: Self) -> Self {
        Align(self.0.min(other.0))
    }
}

/// Trait for computing layouts. Implemented by a `TyCtx` extension
/// or a dedicated database, taking `&TyCtx` as input.
pub trait LayoutComputer {
    fn layout_of(&self, ty: Ty) -> Result<Layout, LayoutError>;
    fn fn_abi_of(&self, sig: &FnSig) -> Result<FnAbi, LayoutError>;
    fn ptr_size(&self) -> Size;
    fn ptr_align(&self) -> Align;
}

#[derive(Clone, Debug)]

    UnknownType(Ty),
    SizeOverflow(Ty),
    Unsized(Ty),
    Cycle(Ty),
}

// ── SimpleLayoutComputer ──

/// A basic layout computer that handles primitives.
/// Production use would extend this with target-specific logic.
pub struct SimpleLayoutComputer<'a> {
    ctx: &'a TyCtx,
    ptr_size: Size,
    ptr_align: Align,
}

impl<'a> SimpleLayoutComputer<'a> {
    pub fn new(ctx: &'a TyCtx, ptr_size: Size, ptr_align: Align) -> Self {
        Self { ctx, ptr_size, ptr_align }
    }
}

impl LayoutComputer for SimpleLayoutComputer<'_> {
    fn layout_of(&self, ty: Ty) -> Result<Layout, LayoutError> {
        match self.ctx.ty_kind(ty) {
            TyKind::Bool => Ok(Layout::scalar(Size::bytes(1))),
            TyKind::Int(i) => Ok(Layout::scalar(Size::bytes(i.bit_width() as u64 / 8))),
            TyKind::Uint(u) => Ok(Layout::scalar(Size::bytes(u.bit_width() as u64 / 8))),
            TyKind::Float(f) => Ok(Layout::scalar(Size::bytes(f.bit_width() as u64 / 8))),
            TyKind::Char => Ok(Layout::scalar(Size::bytes(4))),
            TyKind::Never => Ok(Layout::scalar(Size::ZERO)),
            TyKind::Unit => Ok(Layout::unit()),
            TyKind::Ref(_, _, _) | TyKind::RawPtr(_, _) => {
                Ok(Layout::scalar(self.ptr_size))
            }
            TyKind::Slice(_) | TyKind::Dynamic(_, _) => {
                Err(LayoutError::Unsized(ty))
            }
            TyKind::Error => Err(LayoutError::UnknownType(ty)),
            _ => Err(LayoutError::UnknownType(ty)), // Stub for v0.1.0
        }
    }

    fn fn_abi_of(&self, sig: &FnSig) -> Result<FnAbi, LayoutError> {
        let args = self.ctx.substitution_args(sig.inputs);
        let arg_abis: Vec<ArgAbi> = args
            .iter()
            .filter_map(|arg| {
                if let GenericArg::Ty(t) = arg {
                    Some(ArgAbi {
                        ty: *t,
                        layout: self.layout_of(*t).ok()?,
                        mode: PassMode::Direct,
                    })
                } else {
                    None
                }
            })
            .collect();

        let ret_layout = self.layout_of(sig.output)?;

        Ok(FnAbi {
            args: arg_abis,
            ret: ArgAbi {
                ty: sig.output,
                layout: ret_layout,
                mode: PassMode::Direct,
            },
            conv: CallConvention::from(sig.abi),
            c_variadic: sig.c_variadic,
        })
    }

    fn ptr_size(&self) -> Size {
        self.ptr_size
    }

    fn ptr_align(&self) -> Align {
        self.ptr_align
    }
}

    // ── Unification ──

    #[tracing::instrument(skip(self, ctx))]
    pub fn unify(
        &mut self,
        ctx: &mut TyCtxMut,
        a: Ty,
        b: Ty,
    ) -> Result<Vec<Constraint>, Vec<GlyimDiagnostic>> {
        let a = self.resolve_ty_shallow(ctx, a);
        let b = self.resolve_ty_shallow(ctx, b);
        self.unify_tys(ctx, a, b)
    }

    fn unify_tys(
        &mut self,
        ctx: &mut TyCtxMut,
        a: Ty,
        b: Ty,
    ) -> Result<Vec<Constraint>, Vec<GlyimDiagnostic>> {
        // If both point to the same interned type, they're equal.
        if a == b {
            return Ok(Vec::new());
        }

        let a_kind = ctx.ty_kind(a).clone();
        let b_kind = ctx.ty_kind(b).clone();

        match (a_kind, b_kind) {
            // Infer vars: bind one to the other
            (TyKind::Infer(InferVar::Ty(var)), _) => {
                self.type_variables[var].value = Some(b);
                Ok(Vec::new())
            }
            (_, TyKind::Infer(InferVar::Ty(var))) => {
                self.type_variables[var].value = Some(a);
                Ok(Vec::new())
            }

            // Structural equality for constructors
            (TyKind::Ref(r_a, ty_a, mut_a), TyKind::Ref(r_b, ty_b, mut_b)) => {
                if mut_a != mut_b {
                    return Err(vec![GlyimDiagnostic::type_error(
                        glyim_span::Span::DUMMY,
                        format!("mismatched mutability: {:?} vs {:?}", mut_a, mut_b),
                    )]);
                }
                let mut constraints = Vec::new();
                constraints.push(Constraint::RegionEq { a: r_a, b: r_b });
                let ty_constraints = self.unify_tys(ctx, ty_a, ty_b)?;
                constraints.extend(ty_constraints);
                Ok(constraints)
            }

            (TyKind::Slice(ty_a), TyKind::Slice(ty_b)) => {
                self.unify_tys(ctx, ty_a, ty_b)
            }

            (TyKind::Error, _) | (_, TyKind::Error) => {
                // Error types unify with everything to suppress cascading errors.
                Ok(Vec::new())
            }

            (a_k, b_k) => {
                Err(vec![GlyimDiagnostic::type_error(
                    glyim_span::Span::DUMMY,
                    format!("mismatched types: {:?} vs {:?}", a_k, b_k),
                )])
            }
        }
    }

    // ── Resolution ──

    /// Resolve a type as far as possible without forcing anything.
    pub fn resolve_ty_shallow(&self, ctx: &TyCtxMut, ty: Ty) -> Ty {
        let kind = ctx.ty_kind(ty);
        match kind {
            TyKind::Infer(InferVar::Ty(var)) => {
                if let Some(value) = self.type_variables.get(*var) {
                    if let Some(resolved) = &value.value {
                        return self.resolve_ty_shallow(ctx, *resolved);
                    }
                }
                ty
            }
            _ => ty,
        }
    }

    /// variables remain unresolved.
    pub fn fully_resolve(&self, ctx: &TyCtxMut, ty: Ty) -> Result<Ty, Vec<TyVar>> {

        if ctx.ty_has_infer(resolved) {

            Err(vec![]) // Stub: walk the type to collect vars

    }

    }

    /// Get the current call stack spans for [D2] backtrace.
    fn call_stack_spans(&self) -> Vec<Span> {
        self.stack.iter().map(|f| f.current_span).collect()
    }

    /// Interpret a MIR body.
    ///
    /// `bodies` is a closure that resolves function definitions
    /// when the interpreter encounters a `Call` terminator.
    #[tracing::instrument(skip(self, body, args, bodies))]
    pub fn interpret(
        &mut self,
        body: &Arc<Body>,
        args: &[InterpValue],
        bodies: &dyn Fn(FnDefId) -> Option<Arc<Body>>,
    ) -> InterpResult {
        // Check recursion limit
        if self.stack.len() >= self.config.recursion_limit as usize {
            return InterpResult::StackOverflow;
        }

        // Push initial frame
        self.stack.push(Frame::new(body.clone(), args));

        // Main interpretation loop
        loop {
            self.steps += 1;
            if self.steps > self.config.step_limit {
                return InterpResult::TimedOut;
            }

            let frame = match self.stack.last_mut() {
                Some(f) => f,
                None => return InterpResult::Finished(InterpValue::Zst),
            };

            let current_block = frame.current_block;
            let block_data = match frame.body.basic_blocks.get(current_block) {
                Some(b) => b,
                None => {
                    return InterpResult::Panic {
                        message: format!("invalid basic block: {:?}", current_block),
                        span: frame.current_span,
                        call_stack: self.call_stack_spans(),
                    };
                }
            };

            // Execute statements
            if frame.stmt_index < block_data.statements.len() {
                let stmt = &block_data.statements[frame.stmt_index];
                frame.current_span = stmt.source_info.span;
                frame.stmt_index += 1;

                if let Err(e) = self.exec_statement(stmt) {
                    return e;
                }
                continue;
            }

            // Execute terminator
            let terminator = &block_data.terminator;
            frame.current_span = terminator.source_info.span;

            match self.exec_terminator(terminator, bodies) {
                Ok(InterpAction::Continue) => continue,
                Ok(InterpAction::Return(val)) => {
                    self.stack.pop();
                    if self.stack.is_empty() {
                        return InterpResult::Finished(val);
                    } else {
                        // Store return value in the caller's destination
                        let caller = self.stack.last_mut().unwrap();
                        // Stub: store val in caller's destination place
                        let _ = &caller;
                        continue;
                    }
                }
                Err(result) => return result,
            }
        }
    }

    fn exec_statement(&mut self, stmt: &Statement) -> Result<(), InterpResult> {
        match &stmt.kind {
            StatementKind::Assign(place, rvalue) => {
                let _ = (place, rvalue);
                // Stub: evaluate rvalue and write to place
                Ok(())
            }
            StatementKind::StorageLive(_) => Ok(()),
            StatementKind::StorageDead(_) => Ok(()),
            StatementKind::Nop => Ok(()),
        }
    }

    fn exec_terminator(
        &mut self,
        terminator: &Terminator,
        bodies: &dyn Fn(FnDefId) -> Option<Arc<Body>>,
    ) -> Result<InterpAction, InterpResult> {
        match &terminator.kind {
            TerminatorKind::Goto { target } => {
                let frame = self.stack.last_mut().unwrap();
                frame.current_block = *target;
                frame.stmt_index = 0;
                Ok(InterpAction::Continue)
            }

            TerminatorKind::Return => {
                let frame = self.stack.last().unwrap();
                let return_val = frame.locals.first().and_then(|v| v.clone())
                    .unwrap_or(InterpValue::Zst);
                Ok(InterpAction::Return(return_val))
            }

            TerminatorKind::Unreachable => {
                Err(InterpResult::Panic {
                    message: "reached unreachable code".into(),
                    span: terminator.source_info.span,
                    call_stack: self.call_stack_spans(),
                })
            }

            TerminatorKind::Call { func, args, destination, target, cleanup, .. } => {
                // Evaluate function and arguments
                let _ = (func, args, destination, cleanup);
                // Stub: resolve the callee, push a new frame, or call built-in
                Ok(InterpAction::Continue)
            }

            TerminatorKind::SwitchInt { discr, targets, .. } => {
                let _ = discr;
                let frame = self.stack.last_mut().unwrap();
                // Stub: evaluate discriminant, pick target
                frame.current_block = targets.otherwise();
                frame.stmt_index = 0;
                Ok(InterpAction::Continue)
            }

            TerminatorKind::Assert { cond, expected, target, msg, .. } => {
                let _ = (cond, expected, msg);
                let frame = self.stack.last_mut().unwrap();
                // Stub: evaluate condition, check assertion
                frame.current_block = *target;
                frame.stmt_index = 0;
                Ok(InterpAction::Continue)
            }

            TerminatorKind::Drop { place, target, .. } => {
                let _ = place;
                let frame = self.stack.last_mut().unwrap();
                frame.current_block = *target;
                frame.stmt_index = 0;
                Ok(InterpAction::Continue)
            }
        }
    }
}

/// Internal action returned by terminator execution.
enum InterpAction {
    Continue,
    Return(InterpValue),
}

        }

        if sink.has_errors() {

        // ── Phase 9: Codegen [C1] ──
        let backend = glyim_codegen_llvm::LlvmBackend::new();
        backend.generate(
            ty_ctx,
            &optimized_bodies,
            std::path::Path::new("output.o"),
        )?;

        Ok(())
    }
}

//!
//! This crate provides the runtime support that compiled Glyim
//! programs link against. For v0.1.0, this is minimal.

use std::alloc::{self, Layout};
use std::ptr;

// ── Memory allocation ──

/// Allocate memory with the given size and alignment.
/// Returns a null pointer on failure.
#[no_mangle]
pub extern "C" fn glyim_alloc(size: usize, align: usize) -> *mut u8 {
    unsafe {
        let layout = match Layout::from_size_align(size, align) {
            Ok(l) => l,
            Err(_) => return ptr::null_mut(),
        };
        alloc::alloc(layout)
    }
}

/// Deallocate memory previously allocated with `glyim_alloc`.
#[no_mangle]
pub extern "C" fn glyim_dealloc(ptr: *mut u8, size: usize, align: usize) {
    unsafe {
        if let Ok(layout) = Layout::from_size_align(size, align) {
            alloc::dealloc(ptr, layout);
        }
    }
}

/// Reallocate memory.
#[no_mangle]
pub extern "C" fn glyim_realloc(
    ptr: *mut u8,
    old_size: usize,
    old_align: usize,
    new_size: usize,
) -> *mut u8 {
    unsafe {
        let old_layout = match Layout::from_size_align(old_size, old_align) {
            Ok(l) => l,
            Err(_) => return ptr::null_mut(),
        };
        alloc::realloc(ptr, old_layout, new_size)
    }
}

#[no_mangle]

# Contract Governance Rules

These rules are binding for all contributors. Violations require a Contract Change Request (CCR).

1. **`TyCtx` is the spine.** No logic crate bypasses `TyCtx` to resolve types. Post-typeck, `TyCtx` is frozen and passed as `&TyCtx`.

2. **No `&mut TyCtx` beyond typeck.** If a review finds `&mut TyCtx` in lowering, borrowck, or codegen, it is a **critical bug**.

3. **No `Box`/`Vec` in Chalk interner.** All `Interned*` types are `u32`. If a review finds `Box<chalk_ir::...>`, it is a **critical bug**.

4. **Tracing on all public APIs.** Every public function in a logic crate must carry `#[tracing::instrument]`.

5. **Changes require a CCR.** Changes to public types require a written "Contract Change Request" reviewed by at least one other track lead.

6. **No `unsafe` without a safety proof.** Every `unsafe` block must have a `// SAFETY:` comment explaining why it is sound.

7. **Primitives isolation.** Frontend crates (`glyim-syntax`, `glyim-hir`, `glyim-lex`, `glyim-parse`) must never depend on `glyim-type`.

8. **Hygiene encapsulation.** Only `glyim-hygiene` may call `HygieneKey::new_unchecked()`. Any other caller is a **critical bug**.

9. **Chalk scratchpad scoping.** `GlyimChalkInterner` raw pointer must only be dereferenced during `typeck_crate`. Any access after `freeze()` is a **critical bug**.

10. **Error type propagation.** The `TyKind::Error` type must propagate through type operations without causing panics or additional errors.
