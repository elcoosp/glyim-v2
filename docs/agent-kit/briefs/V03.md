# Stream SV03: Default Methods

## Mission
Implement Allow trait methods to have default implementations. Implement inheritance: impls can override default methods or inherit them. Ensure correct dispatch and that default methods can call other trait methods (including default ones). for the Glyim compiler.

- Add dev-dependencies for testing

## Test Location (MANDATORY)

**ALL tests MUST be written as unit tests in `src/tests/` within the crate.**

NEVER use the `tests/` directory at the crate root (integration tests). Unit tests compile faster because they share the crate's private API.

### Setup Pattern

1. In `crates/glyim-typeck/src/lib.rs`, add:
   ```rust
   #[cfg(test)]
   mod tests;
   ```

2. Create `crates/glyim-typeck/src/tests/mod.rs` declaring submodules:
   ```rust
   mod module_a;
   mod module_b;
   ```

3. Each test file contains `#[test]` functions with full access to `pub(crate)` items.

4. Add `glyim-test` as a dev-dependency:
   ```toml
   [dev-dependencies]
   glyim-test = { workspace = true }
   ```

5. Read `docs/agent-kit/GLYIM_TEST_INSTRUCTIONS.md` for full API reference.
